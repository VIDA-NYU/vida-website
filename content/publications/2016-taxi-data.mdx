TaxiVis is a visual analytics system for exploring taxi trip data. The system enables users to interactively query and visualize millions of taxi trips, supporting the discovery of patterns in urban mobility.

## Key Features

- Interactive querying of large-scale taxi trip datasets
- Visual exploration of spatial and temporal patterns
- Support for comparative analysis across different time periods
- Integration with urban infrastructure data
