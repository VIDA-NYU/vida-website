VisFlow is a web-based dataflow framework for visual data exploration. It employs a subset dataflow that allows the user to interactively select, manipulate, brush and link data subsets across multiple visualizations.

## Features

- **Subset Dataflow**: Interactively select and manipulate data subsets
- **Brushing and Linking**: Connect visualizations across multiple views
- **Web-Based**: Run analyses directly in a web browser
- **Simple and Intuitive**: Easy to learn and use

## Benefits

VisFlow is simple and intuitive to use and can help fast launch data analyses in a web browser. It focuses on interactivity, flexibility, and simplicity.
