The Data-Driven Discovery of Models (D3M) program aims to develop automated model discovery systems that enable users with subject matter expertise but no data science background to create empirical models of real, complex processes.

## Key Components

- **AlphaD3M**: Machine learning pipeline synthesis using reinforcement learning
- **d3m-interface**: Python library for integrating D3M AutoML systems with Jupyter Notebooks
- **PipelineProfiler**: Visual analytics tool for exploring AutoML pipelines
